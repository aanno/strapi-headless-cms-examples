Run with:

```bash
$ yarn develop
...
To access the server ⚡️, go to:
http://localhost:1337
```

### Changes

* Use yarn 2 (3.2.0) in 'zero-install' with `pnp` linker
* Use strapi 4 (instead of strapi 3)
